# Keystone Species & Trophic Cascades

An interactive web lesson for 10th grade general biology, built for Mr. Nebbe's classes.

## Topics covered
- What a keystone species is
- How trophic cascades work
- The Aleutian kelp forest (orcas → sea otters → sea urchins → kelp)
- Dr. Robert Paine's tidal pool experiment with *Pisaster* sea stars

## Features
- Two interactive ecosystem simulators ("remove the predator" and watch the cascade)
- English / Spanish language toggle for ELL students
- Floating glossary panel with 16 bilingual terms
- Concept check: 6 multiple-choice questions + 1 essay
- Compile / copy button — students paste answers into Google Classroom

## Based on
HHMI BioInteractive video: [Some Animals Are More Equal than Others — Keystone Species and Trophic Cascades](https://www.youtube.com/watch?v=hRGg5it5FMI)
